A WordPress plugin to task manager for administrator

== Description ==

This plugin is a simple version for creating tasks and managing tasks for admins and authors of your site

== Installation ==

1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.

== Frequently Asked Questions ==

= Why should I use this plugin? =

The easiest way to make simple tasks

= How to uninstall the plugin? =

Simply deactivate and delete the plugin.

== Screenshots ==
1. Modal view for create new task and show your active task.
2. Sub menu for show all tasks created.
